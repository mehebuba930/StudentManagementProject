# Student Management System

## Project Overview
Student Management System is a Java-based console application developed using Object-Oriented Programming (OOP) concepts. The system manages student records, teacher information, academic data, and user accounts efficiently.

## Features

### Student Management
- Add Student
- Update Student Information
- Delete Student
- Search Student
- Display All Students
- Sort Students by GPA
- Sort Students by Marks

### Teacher Management
- Add Teacher
- Update Teacher Information
- Delete Teacher
- Display Teacher Information

### Academic Management
- Department Management
- Degree Level Management
- Curriculum Catalog Management
- Course Result Management

### Account Management
- User Login
- User Authentication
- Account Storage and Retrieval

### File Handling
- Save Student Data to File
- Load Student Data from File
- Save Account Information

## OOP Concepts Used
- Class and Object
- Inheritance
- Encapsulation
- Abstraction
- Polymorphism
- File Handling
- Collections Framework (ArrayList)

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File I/O
- Collections Framework
- VS Code

## How to Run
1. Open the project in VS Code or any Java IDE.
2. Compile all Java files.
3. Run `Main.java`.
4. Follow the console menu instructions.
